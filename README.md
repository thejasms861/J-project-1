# Smart Stadium Experience Platform 🏟️

Welcome to the Smart Stadium Experience Platform repository.

## Structure
- `frontend-user/`: Next.js Mobile-first web app for fans
- `frontend-admin/`: Vite React SPA for admin and ops dashboards
- `backend/`: FastAPI backend with PostgreSQL & Redis
- `ai-engine/`: AI Recommendation & Analytics pipelines
- `infra/`: Nginx and Prometheus configs
- `docs/`: System design blueprints and API specs

## Running Locally
```bash
docker-compose up -d
```
